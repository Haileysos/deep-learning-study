# Binary Crossentropy (이진 교차 엔트로피)  
: 이진 분류 문제를 해결하는 데 사용되는 손실 함수  
- 분류해야 할 부류가 두 가지 뿐일 때 사용  
- e.g. 강아지 이미지를 강아지O/강아지X 의 두 부류로 분류할 때

# Categorical Crossentropy   
- 분류해야 할 부류가 두 개 이상일 때(다중 분류 문제일 때) 사용  
- 정답 레이블은 **원-핫 인코딩**으로 제공  
- e.g. 입력 이미지를 “강아지(1,0, 0)”, “고양이(0, 1, 0)”, “호랑이(0, 0, 1)” 중의 하나로 분류하는 경우 사용

# SparseCategorical Crossentropy  
- 정답 레이블이 원-핫 인코딩이 아니고 정수로 주어질 때 사용
- e.g. 정답 레이블이 0(강아지), 1(고양이), 2(호랑이)로 주어진 경우 사용
