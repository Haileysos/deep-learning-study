# 그래디언트 소실 문제  
: DNN(심층 신경망)에서 그래디언트가 전달되다가 점점 0 에 가까워지는 현상

### 범인
- 범인은 시그모이드 활성화 함수로 드러남  
- 시그모이드 함수의 특성상, 아주 큰 양수나 음수가 들어오면 출력이 포화되어 그래디언트가 거의 0이 됨
