# Loss Function (손실함수) 계산  
: 신경망에서 학습을 시킬 때는 실제 출력과 원하는 출력 사이의 오차를 이용하는데!  

이 오차를 계산하는 함수가 **손실함수**  

- 또한, 학습의 성과를 나타내는 지표이기도 함
